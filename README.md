# 💜 Birthday Card App — Paps 2026

A swipeable interactive birthday card experience built with SvelteKit and Tailwind.

This project presents memories as full-screen cards:

- Swipe left/right → flip photo & message
- Swipe up/down → next/previous memory
- Purple themed captions & messages
- Mobile-first immersive layout

---

## ✨ Features

- 3D flip card animation
- Gesture navigation (horizontal + vertical)
- Responsive full-screen images (no distortion)
- Purple birthday theme
- Smooth transitions
- Mobile-optimized UX

---

## 🛠 Tech Stack

- SvelteKit 5
- TypeScript
- Tailwind CSS v4
- pnpm
- Vite

---

## 🚀 Run locally

```bash
pnpm install
pnpm dev
```

### ✅ Now rebuild & publish

```bash
rm -rf build
pnpm build
npx gh-pages -d build --dotfiles --no-history
```
